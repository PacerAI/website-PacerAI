# Project Overview — Pacer AI Homepage Refresh

**Sprint:** Website Refresh v1  
**Date:** March 2026  
**Owner:** Will Sullivan, Founder — Pacer AI  
**Status:** In Progress

---

## Goal

Deploy a redesigned homepage for getpacerai.com that clearly communicates Pacer AI's value proposition to PE-backed SaaS operators and converts visitors into demo requests.

## Problem Statement

The current homepage does not reflect Pacer AI's positioning as a board-level ARR intelligence platform. It lacks the financial-professional aesthetic and specific use case clarity needed to resonate with Operating Partners, CFOs, and RevOps leaders evaluating operational data tools.

## Success Criteria

- [ ] Redesigned homepage live at getpacerai.com
- [ ] Primary CTA ("Request a Demo") above the fold
- [ ] ARR Snowball use case clearly featured as primary product story
- [ ] Navigation matches target structure (Platform, Solutions, Resources, Customers, Company)
- [ ] Lighthouse Performance score ≥ 85
- [ ] Lighthouse SEO score ≥ 90
- [ ] Mobile-responsive on iPhone 14 and iPad Pro viewports
- [ ] Yoast SEO metadata intact and updated

## Scope — In

- Homepage only (`/`)
- Navigation menu structure update
- Footer update (Platform, Products, Use Cases, Comparisons, Company)

## Scope — Out

- Blog, product pages, or any other existing pages
- New media uploads (use existing assets where possible)
- Backend/plugin changes
